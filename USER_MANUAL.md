# USER MANUAL

## 1. 用途

Karuta Jukebox 用于组织“听歌猜作品”对局，不是通用播放器。  
核心能力：数据集维护、开局选卡、回合控制、后台管理。

## 2. 主界面

主界面提供：

- 数据集选择与预览
- 对局参数设置（卡牌上限、失败处理、休息音乐）
- 启动对局
- 打开数据集导入/编辑器

## 3. 数据集编辑器

### 3.1 基本操作

1. 选择或新建数据集。
2. 新建作品：选择图片 + 选择歌曲 + 填写显示名 + 保存。
3. 编辑作品：左侧选中作品，修改后保存覆盖。

### 3.2 flac 导入

- 导入时允许选择 `flac`。
- 保存时调用 `ffmpeg` 转为 `mp3`。
- CSV 最终写入的是转码后的文件名。

### 3.3 数据包

- 支持将数据集导出为 ZIP（CSV + 关联资源）。
- 支持从 ZIP 导入并自动处理同名冲突。
- 删除数据集支持：
- 仅删除 CSV
- 删除 CSV + 独占资源文件

## 4. 开局与选卡

1. 点击 `Start` 后先进入选卡窗口，不会直接开始。
2. 系统会自动预选前 N 张（N 为主界面卡牌上限）。
3. 你可手动增减，最终以确认时勾选结果为准。
4. 未参赛卡牌的歌曲可进入休息音乐池。

## 5. 对局流程

1. 进入对局后先点 `Prepare`。
2. 回合中可随时点 `Success` / `Failure`。
3. 回合结束进入休息状态。
4. 点 `Prepare Next` 开始下一回合。

`On Failure` 说明：

- `PASS`：失败后卡牌保留在 active 池。
- `SKIP`：失败后卡牌移出 active 池。

## 6. 管理员面板

管理员面板可实时查看 active/inactive 卡牌，支持：

- 将选中卡牌移动到 active
- 将选中卡牌移动到 inactive
- 导出当前状态文本
- 重置 inactive 回 active

## 7. 常见问题

- 中文 CSV 打开乱码：建议通过程序保存一次，输出为 UTF-8 BOM。
- 音频不播放：检查扩展名是否在支持列表中。
- 找不到配置：确认运行目录下存在可用 `config/config.txt`。

# GET STARTED

这份文档面向第一次使用项目的人，按“准备素材 -> 建数据集 -> 开局”顺序说明。

## 1. 理解基本模型

一个“作品”包含：

- 1 张图片（卡面）
- 作品标题
- 1~N 首歌曲
- 每首歌的显示名

一个“数据集”对应一个 CSV 文件，内部包含多个作品。

## 2. 准备素材

将素材放到运行目录的 `config` 下：

- 图片：`config/images/`
- 音频：`config/music/`
- 数据集：`config/decks/`

建议音频优先使用 `mp3`。  
如果只有 `flac`，可以在导入界面选择，保存时会自动转为 `mp3`。

## 3. 启动程序

```bash
mvn clean javafx:run
```

## 4. 创建或编辑数据集

1. 在主界面点击“导入数据集”。
2. 选择已有数据集，或输入新数据集名称。
3. 点击“新建作品”进入录入模式。
4. 依次选择图片、歌曲并填写显示名。
5. 点击“保存作品”。
6. 重复以上步骤直到录入完成。

你也可以在左侧作品列表点选已有作品并修改后覆盖保存。

## 5. 开始一局

1. 回到主界面，选择数据集。
2. 设置卡牌数量上限和失败处理方式。
3. 点击 `Start`。
4. 在选卡窗口确认本局参赛卡牌。
5. 进入游戏后点击 `Prepare`，开始第一回合。

## 6. 回合节奏

1. 音频播放中随时可点 `Success` / `Failure`。
2. 回合结算后进入休息状态。
3. 点击 `Prepare Next` 进入下一回合。

注意：第一回合也必须手动 `Prepare`，不会自动开始。
